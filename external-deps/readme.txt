To get started, go to https://developer.vuforia.com/downloads/sdk, where you will find the Vuforia-iOS SDK. Download the sdk, and unpack the .zip file to the `Vuforia` folder.
As for vuforia-sdk-ios-5-5-9, the structure should be like:

external-deps/
	readme.txt
	Vuforia/
		assets/
		build/
		licenses/
		readme.txt
		samples/
